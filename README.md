# CNN Accelerator using Systolic Array

## 📌 Overview
Designed a hardware accelerator for CNN inference using systolic array architecture and ping-pong buffering.

## 🚀 Features
- High-throughput systolic PE array
- Efficient memory usage with ping-pong buffers
- DMA + AXI for optimized data movement

## 🏗️ Architecture
- Processing Element (PE) Array
- On-chip Buffer
- DMA Engine
- AXI Interface

## ⚙️ Tech Stack
- Verilog / SystemC (HLS)
- Xilinx FPGA

## 📊 Results
- Throughput: 0.5 TOPS
- Power: 3.02 W
